# 🌤 Weather-Cast

Weather-Cast is a modern weather application that provides real-time weather information using the OpenWeatherMap API. Users can search any city and get live temperature, weather condition, humidity, and weather icons.

## 🌐 Live Demo

Visit the deployed application:

https://sprint-3-virid.vercel.app/


## 🚀 Features

- 🌍 Search weather by city name
- 🌡️ Real-time temperature updates
- ☁️ Weather condition display
- 💧 Humidity information
- 🌦 Dynamic weather icons
- 🖼️ Background weather image
- ⚡ Fetch API integration
- 🔄 Async/Await JavaScript
- ❌ Error handling for invalid cities
- 📱 Responsive user interface


## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- OpenWeatherMap API
- Vercel

## 🔑 API Setup

This project uses OpenWeatherMap API for fetching live weather data.

Steps:

1. Create an account on OpenWeatherMap
2. Generate your API key
3. Add your API key in script.js

Example:

const API_KEY = "YOUR_API_KEY";
